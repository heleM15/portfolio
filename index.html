Meus projetos

Projeto Monaliza

No projeto Monaliza, eu recreiei uma personagem que representa o efeito de ótica do quadro da Monaliza aonde os olhos parecem acompanhar quem está visualizando essa imagem, isso é possível através do comando map, para com o que as pupilas do personagem acompanhem o mouse.

Projeto Quente-Frio

Neste projeto, foi feito um jogo de quebra-cabeça com o objetivo de encontrar no canvas uma pequenas bola de 3-1 pixel de tamanho em uma tela escura, para guiar o jogador, envolta do mouse tem um circulo que diminui ou aumenta depedendo da proximidade do pixel, ele é colocado de forma aleatória pelo comando "random".

Projeto Corrida

Nesse projeto o objetivo foi aprender a otimizar o sistema de listas do p5.js para representar jogadores em uma corrida.
